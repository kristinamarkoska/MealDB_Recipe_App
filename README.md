# Recipe Explorer App

## Getting Started
The app allows users to browse meal categories, view meals by category, search for meals, and explore detailed recipe information.
Features

Display all meal categories in a card layout

Search categories

View meals by selected category (grid layout)

Search meals within a category

Detailed recipe view including:

Image

Name

Instructions

Ingredients

YouTube video link (if available)

Random recipe of the day feature

API Used

Categories:
https://www.themealdb.com/api/json/v1/1/categories.php

Meals by category:
https://www.themealdb.com/api/json/v1/1/filter.php?c={category}

Search meals:
https://www.themealdb.com/api/json/v1/1/search.php?s={query}

Meal details:
https://www.themealdb.com/api/json/v1/1/lookup.php?i={id}

Random meal:
https://www.themealdb.com/api/json/v1/1/random.php
